Quantsus: A Python package for Sus's quant analyses
=======================================

Quantsus.


Installation
-------------

Please install quantsus with the following command:

`pip install quantsus`


Dependencies
------------

Quantsus supports Python 3.8+.

Installation requires [numpy](https://numpy.org/), [pandas](https://pandas.pydata.org/), [matplotlib](https://matplotlib.org/), [seaborn](https://seaborn.pydata.org/), [statsmodels](https://statsmodels.org/), [yfinance](https://github.com/ranaroussi/yfinance), and [scipy](https://scipy.org/).

Citation
------------

Quantsus is not a published work, so citing the GitHub repo suffices. Please refer to the `CITATION.cff` file.
